# Mini Projet Architecture SI

## Description du projet

Le projet consistait à choisir et concevoir un **système d'information (SI)** non-existant (nommé **Loc'Info**) afin d'en présenter les différentes couches d'architecture. Le SI conçu est une solution de location de matériel informatique pour les étudiants.

Travail de groupe réalisé à deux.

## Outil utilisé
- Cours d'architecture d'un système d'informations

## Sommaire
- **Cahier des Charges Techniques - William Andres.pdf** : Cahier des Charges Techniques.
