# Mini Projet Architecture SI

## Description du projet

Le projet consistait à choisir et concevoir un **système d'information (SI)** non-existant (nommé **Loc'Info**) afin d'en présenter les différentes couches d'architecture. Le SI conçu est une solution de location de matériel informatique pour les étudiants.

Le cahier des charges de ce système d'information, intitulé "**Cahier des Charges Techniques - William Andres.pdf**", est disponible en format PDF dans le dossier principal de ce dépôt.

Travail de groupe réalisé à deux.

## Outil utilisé
- Cours d'architecture d'un système d'informations

## Sommaire
- **Cahier des Charges Techniques - William Andres.pdf** : Rapport détaillé de l'architecture du SI **Loc'Info**. Ce document inclut :
    - La présentation générale
    - Le schéma UML de l'application
    - Les choix technologiques
    - La projection dans une architecture client-serveur
    - Les fonctionnalités techniques
    - Les trois visions
